# Canvas 平面直角坐标系(Rectangular Plane Coordinate System)

## 在上面绘制简单的几何图形(Draw some simple geometric figure)

## 编写自己的图形的方法
* initGrid(); 可选是否初始化平面直角坐标系
* drawHouse(); 示例，绘制房间图
* drawGeometricFigure(list, bool); list提供数据，bool 提供缩放功能。
